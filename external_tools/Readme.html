DDE-Biftool external tools


The subfolder support scripts in languages other than Matlab/octave.
  • genr_sys.mth A Mathematica script to generate the right-hand side and its derivatives (sys_deri.m) for use in DDE-BIFTOOL.
  • Maple_gen_sys.mw A Maple script to generate the right-hand side and its derivatives (including sys_deri.m and, for normalform computations, sys_mfderi.m) for use in DDE-BIFTOOL.
